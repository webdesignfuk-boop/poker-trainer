# 🃏 テキサスホールデム ポーカートレーナー

リアル志向のテキサスホールデムポーカーゲームで、AI と対戦しながらポーカースキルを向上させるトレーニングアプリです。

## 🎯 主要機能

### 1. リアルなゲームプレイ
- **プレイヤー 1人 vs AI 3人**
- **3つの異なる AI プレイスタイル**:
  - **AI Tight**: 保守的なプレイスタイル（参加率20%）
  - **AI Loose**: 積極的に参加するスタイル（参加率45%）
  - **AI Aggressive**: 攻撃的なプレイスタイル（レイズ頻度60%）
- **フルラウンド実装**: プリフロップ → フロップ → ターン → リバー
- **リアルなベッティングシステム**: チェック、コール、レイズ、フォールド

### 2. 詳細なデータ記録
- 全てのハンドの完全な履歴
- プレイヤーとAIの全アクション記録
- AIの判断理由も記録
- ベッティングパターンの追跡

### 3. 高度なフィードバック機能
- **ゲーム後の詳細分析**
- **良かったプレイの指摘**
- **改善が必要なプレイの指摘**
- **具体的な改善提案**
- **統計データの表示**:
  - VPIP（参加率）
  - PFR（プリフロップレイズ率）
  - 勝率
  - 総利益/損失
  - フォールド率

### 4. リアルなWeb UI
- カジノテーブル風のデザイン
- 美しいカード表示
- チップとポットの可視化
- リアルタイムゲームログ
- インタラクティブなベッティング操作

## 🚀 起動方法

### 必要環境
- Python 3.8以上
- Flask

### インストールと起動

1. **依存関係のインストール**:
```bash
pip install flask
```

2. **サーバー起動**:
```bash
python server.py
```

3. **ブラウザでアクセス**:
```
http://localhost:5000
```

## 🎮 遊び方

1. **「新しいハンドを開始」ボタンをクリック**
2. **自分のカードを確認**
3. **アクションを選択**:
   - **フォールド**: ハンドを降りる
   - **チェック**: ベット額が0の時、パス
   - **コール**: 現在のベット額に合わせる
   - **レイズ**: ベット額を上げる（スライダーで金額調整）
4. **AIが順番に行動**
5. **各ストリート（フロップ、ターン、リバー）を進行**
6. **ショーダウンで勝者決定**
7. **ゲーム後にフィードバックを確認**

## 📊 フィードバックシステム

### 統計情報
- **総ハンド数**: プレイした全ハンド数
- **VPIP**: プリフロップで参加した割合
- **PFR**: プリフロップでレイズした割合
- **勝率**: ショーダウンでの勝利率
- **総利益**: 累積の利益/損失

### プレイ分析
- **良かったプレイ**: 戦略的に正しい判断
- **改善が必要なプレイ**: 最適でなかった判断
- **アドバイス**: 具体的な改善提案

### 総合評価
- プレイスタイルの分析
- 強みと弱みの特定
- パーソナライズされた推奨事項

## 🤖 AIの特徴

### AI Tight（タイト）
- 参加率: 20%
- レイズ頻度: 30%
- ブラフ頻度: 5%
- 特徴: 強いハンドのみでプレイ

### AI Loose（ルース）
- 参加率: 45%
- レイズ頻度: 25%
- ブラフ頻度: 15%
- 特徴: 多くのハンドに参加

### AI Aggressive（アグレッシブ）
- 参加率: 35%
- レイズ頻度: 60%
- ブラフ頻度: 25%
- 特徴: 攻撃的なプレイ、頻繁なレイズ

## 🎓 学習のポイント

1. **ハンド選択**: 強いハンドを選んで参加
2. **ポジション**: 後ろのポジションほど有利

5. **相手の傾向**: AIのプレイスタイルを観察
6. **ブラフ**: タイミングを見てブラフを仕掛ける
